# claude-code-skills

Ne vous méprenez pas, ce projet est mal nommé !

Mes compétences Claude dans /skills, /commands et /rules, par compétences j'entends "mes idées" de skills, agents, commandes et autres qui permettent d'implanter une méthode SDD dans une organisation.

Les skills SDD en sdd-uc-<action> sont des skills utilisant la méthode des cas d'utilisation (Use Case).
Les skills SDD en sdd-<action> n'utilisent pas la méthode des cas d'utilisation.

Les skills CC sont  conformes à https://agentskills.io/home

Le projet contient également un fichier CLAUDE.md dans /claude-file et qui est le modèle décrivant la boucle de rétroaction SDD. Ce fichier est à utiliser comme modèle en coordination avec les fichiers de skills, agents, commands, rules, etc.

Le fichier CLAUDE.md à la racine est le fichier utilisé pour travailler avec Claude Code à l'amélioration de ce projet.
